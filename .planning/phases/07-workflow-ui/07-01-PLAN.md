## Task Details

### Task 1: `src/workflow.py`

`create_angebotsentwurf(file_path, tenant_id) -> tuple[Path, AngebotData, dict]`

Kompletter Orchestrierungs-Workflow:
1. `load_document()` + `chunk_document()`
2. `extract_requirements(chunks)` → AngebotData
3. `search(summary)` → retrieved_chunks
4. `generate_angebot(data, chunks)` → result dict
5. `write_docx(sections, sources, output_path)` → docx_path

### Task 2: `app.py` Integration

- `_run_workflow_flow()`: AskFileMessage → Extraktion → HIL-Bestätigungs-Schritt → Generierung → Download
- `_deliver_file()`: async Helper für cl.File-Auslieferung
- Nicht-unterstützte Formate (.xlsx) mit deutscher Fehlermeldung abfangen

### Task 3: Tests

Datei: `tests/test_workflow.py`
- Mock load_document, extract_requirements, search, generate_angebot, write_docx
- Prüfe Rückgabe-Tuple (Path, AngebotData, dict)
- Prüfe .docx-Pfad-Format
- Prüfe RAG-Search-Aufruf mit summary
